# PUSH-BUTTON-COUNTER
A TEMPERATURE SENSOR TO READ AND DISPLAY TEMPERATURE DATA ON AN LCD OR SERIAL MONITOR.
# Components Used:
1. Arduino Uno R3
2. LCD 16 x 2
3. 250 kΩ Potentiometer
4. 330 Ω Resistor
5. Temperature Sensor [TMP36]
